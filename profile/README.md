## 👩‍💻 프로젝트 목적 및 SBH IDE 소개

SBH IDE는 자신만의 작업 공간에서 프로젝트를 개발할 수 있는 코드 편집기입니다.  
코드는 서버에 안전하게 저장되므로, 언제든지 로그인만으로 저장된 코드를 불러와 작업을 이어갈 수 있습니다.  
각 ID 별로 분리된 공간에서 사용자는 자유롭게 폴더와 파일을 추가하거나 관리할 수 있습니다.  

</br>

## 👉 특징
1. 로그인
2. 폴더 및 파일 생성
3. 파일 불러오기 및 저장
4. 프로필 사진 및 이름 수정
